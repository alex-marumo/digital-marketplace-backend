# 🎨 Digital Marketplace Backend

This is the backend for **Digital Marketplace for Local Artists**, a platform designed to help artists in Botswana showcase and sell their artwork online.  

## 🚀 Features
- **User Authentication** (JWT-based login & signup)
- **Artwork Management** (Artists can list, update, and delete their artworks)
- **Real-time Messaging** (Buyers and artists can chat via WebSockets)
- **Secure Transactions** (Orders and payments handling)
- **PostgreSQL Database** (Using Sequelize ORM)
- **RESTful API** for frontend integration

## 🛠️ Tech Stack
- **Backend:** Node.js (Express.js)
- **Database:** PostgreSQL (with Sequelize ORM)
- **Authentication:** JSON Web Tokens (JWT)
- **WebSockets:** Socket.io
- **Deployment:** Hosted on Railway.app
